# Homebrew Tap for ja

This is the official Homebrew tap for [ja](https://github.com/toba/skill), an agent multi-tool for little things.

## Installation

```bash
brew install toba/ja/ja
```

## Updating

```bash
brew update
brew upgrade ja
```

## Uninstalling

```bash
brew uninstall ja
brew untap toba/ja
```

## Requirements

- macOS (Apple Silicon)

## Issues

Report issues at [toba/skill](https://github.com/toba/skill/issues).
